## TOOL 1 — COMPANY STAMP / SEAL CREATOR ⭐ (Priority)

This is the most culturally important tool for Arabic-speaking markets. Every business in MENA uses an official round company stamp on documents, contracts, and invoices.

### Features:
- **Canvas-based stamp builder** using HTML5 Canvas or SVG.
- **Shape options**: Round (default), Oval, Square with rounded corners, Rectangle.
- **Stamp layers the user can configure**:
  - Outer ring text (top arc): Company name in Arabic and/or English
  - Inner ring text (bottom arc): City / Country / Registration number
  - Center content: Choose from — Company logo upload, Company initials (auto-styled), Star/emblem icon, or leave blank
  - Optional inner decorative ring (double-ring border style)
  - Optional small star dividers between outer ring text segments
- **Styling controls**:
  - Stamp color: preset palette (classic red, navy blue, green, black, purple, gold) + custom color picker
  - Font size for outer text and inner text (slider)
  - Border width (thin / medium / thick)
  - Ink effect toggle: "Clean" (solid) vs "Ink Stamp" (slightly distressed texture using canvas noise)
- **Live preview**: Full-size preview updates in real time as user changes any setting.
- **Export options**:
  - Download as PNG (transparent background, 500x500px)
  - Download as SVG (vector, scalable)
  - Download as PDF (stamp centered on A4 page)
  - "Copy to clipboard" button
- **Arabic text input**: Must support Arabic font rendering correctly. Use Cairo font for Arabic text in the stamp. Ensure proper letter joining and RTL character rendering on the canvas.
- **Preset stamp styles**: Offer 6 ready-made stamp presets the user can click and customize:
  - Government style (formal, double ring, navy)
  - Modern Corporate (clean single ring, dark blue)
  - Classic Trader (red, bold font, star dividers)
  - Minimalist (thin border, black, no inner ring)
  - Arabic Seal (Arabic text dominant, ornamental center)
  - NGO / Organization (green, oval shape)

### Page layout:
- Left panel: all controls (shape, text inputs, styling)
- Right panel: large live stamp preview on a white card
- Bottom: export buttons

## TOOL 2 — OFFICIAL LETTERHEAD BUILDER

Every professional business needs a branded letterhead. This tool lets users create a reusable company letterhead template.

### Features:
- **A4 page preview** that updates live.
- **Header section** (user configures):
  - Company logo upload (left, center, or right aligned)
  - Company name (large, styled)
  - Tagline / slogan
  - Address block (multi-line)
  - Contact details: phone, email, website
  - Optional: Trade/Commercial Registration number, VAT number
- **Footer section** (user configures):
  - Footer bar color (matches brand color picker)
  - Footer text: address, phone, website in smaller text
  - Optional: social media icons (LinkedIn, Twitter, Instagram, WhatsApp)
- **Design controls**:
  - Primary brand color (updates header bar, footer bar, and accent lines)
  - Header layout style: choose from 3 layouts (logo left + text right | centered logo + centered text | text left + logo right)
  - Divider line style: solid / dashed / double / none
  - Font pairing: Modern (Inter/Cairo) | Classic (serif style) | Bold (heavy heading)
- **Bilingual**: Toggle to show company name in both Arabic and English, stacked.
- **Export**:
  - Download as PDF (blank A4 letterhead ready to print or use as background)
  - Download as PNG
  - "Use as document header" button — saves the letterhead config and applies it as the default header on Invoice, Quotation, and Receipt generators

## TOOL 3 — PURCHASE ORDER (PO) GENERATOR

Critical for procurement, logistics, Oil & Gas, and NGO supply chains.

### Features:
- Same form structure as Invoice Generator but with PO-specific fields:
  - PO Number (auto-generated: PO-2025-001)
  - PO Date + Required Delivery Date
  - Buyer details (your company)
  - Supplier/Vendor details
  - Delivery address (can differ from billing)
  - Line items table: Item code, description, quantity, unit, unit price, total
  - Payment terms
  - Delivery terms (EXW, FOB, CIF, DDP, DAP — dropdown)
  - Special instructions / notes
  - Authorized signature block (name + title)
- **3 PO Templates**:
  1. Standard PO — clean professional format
  2. Formal Government/NGO PO — includes reference fields, department, budget code
  3. Logistics/Freight PO — adds fields for shipment mode (Air/Sea/Road), port of loading, port of discharge, HS code
- PDF export, bilingual, RTL/LTR support.

## TOOL 4 — DELIVERY NOTE GENERATOR

Essential for traders, importers, exporters, and logistics companies.

### Features:
- Fields: Delivery Note number, date, order reference (links to a PO or invoice number), sender details, recipient details, delivery address.
- Items table: item description, quantity delivered, unit, condition (Good / Damaged / Partial).

- **2 Templates**:
  1. Standard Delivery Note
  2. Warehouse Release Note (for warehouse-to-client handover)
- PDF export, bilingual.

## TOOL 5 — PACKING LIST GENERATOR

For import/export businesses and freight forwarders.

### Features:
- Fields: Packing List number, date, shipper, consignee, vessel/flight/truck reference, bill of lading/airway bill number, port of loading, port of discharge.
- Items table: package number, description, quantity, net weight (kg), gross weight (kg), dimensions (L×W×H cm), CBM (auto-calculated).

- **Auto-CBM calculation**: multiply L×W×H for each row, show running total.
- Notes field.
- **2 Templates**:
  1. Standard Packing List
  2. Detailed Cargo Manifest (adds HS code, country of origin, declared value columns)
- PDF export, bilingual.

## TOOL 6 — E-SIGNATURE PAD

Allow users to create a reusable digital signature to insert into their documents.

### Features:
- **3 input methods**:
  1. Draw: canvas-based freehand drawing pad with smooth pressure-simulated stroke
  2. Type: type your name and choose from 6 signature font styles (cursive/script styles)
  3. Upload: upload an existing signature image
- **Controls**: pen color (black, navy, dark blue), stroke thickness (thin/medium/thick), clear button, undo last stroke.
- **Live preview**: show signature on a mock document line.
- **Export**:
  - Download as PNG (transparent background)
  - Download as SVG
  - "Save as my signature" — stores in localStorage and auto-inserts into Invoice, PO, Receipt, and Delivery Note documents at the signature line.
- Bilingual: label the pad in Arabic and English.

## TOOL 7 — QR CODE GENERATOR

High-demand utility tool — great for SEO traffic too.

### Features:
- Input: URL, plain text, phone number (tel:), email (mailto:), WhatsApp link (wa.me/), vCard contact, WiFi credentials, payment link.
- **Smart type detection**: auto-detect input type and suggest the format.
- Customization:
  - QR color: foreground and background color pickers
  - Size: Small (200px) / Medium (400px) / Large (800px)
  - Error correction level: L / M / Q / H
  - Corner style: Sharp / Rounded / Dots
  - Center logo: option to upload a small logo to embed in the center of the QR code
- Live preview updates as user types.
- Export: PNG, SVG, PDF.
- Use the `qrcode` npm library for generation.
- Add a "QR code for this invoice" shortcut button inside the Invoice, Quotation, and Receipt generators — generates a QR pointing to the document's shareable link.

## TOOL 11 — WATERMARK TOOL

Add a professional watermark to any PDF or image document.

### Features:
- Upload a PDF or image (PNG, JPG).
- Watermark options:
  - Text watermark: type custom text (e.g. "CONFIDENTIAL", "DRAFT", "PAID", "COPY", or custom text in Arabic or English)
  - Image watermark: upload a logo
- Controls: opacity slider (10%–80%), rotation (-45° to 45°), position (diagonal / centered / tiled), font size, color.
- Preset quick-stamps: DRAFT, CONFIDENTIAL, PAID, VOID, SAMPLE, COPY — one-click apply.
- Preview: show the document with watermark overlay before downloading.
- Export: download as PDF or PNG with watermark applied.
