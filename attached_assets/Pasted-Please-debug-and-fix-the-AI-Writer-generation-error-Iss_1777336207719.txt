3. Standardize the backend response format

For successful AI generation, return:

{
  "success": true,
  "subject": "Generated subject here",
  "body": "Generated message body here",
  "provider": "gemini"
}

{
  "success": false,
  "error": "AI provider unavailable",
  "message": "AI generation is not active yet. Please configure the Gemini API key."
}

{
  "success": false,
  "error": "Invalid request",
  "message": "Please complete the required fields before generating."
}

For AI provider errors, return HTTP 502 with:

{
  "success": false,
  "error": "AI provider error",
  "message": "The AI provider could not generate the message. Please try again."
}

For unexpected errors, return HTTP 500 with:

{
  "success": false,
  "error": "Server error",
  "message": "Something went wrong while generating the message."
}
