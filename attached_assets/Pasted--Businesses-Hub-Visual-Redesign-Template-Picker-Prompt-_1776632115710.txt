## PART 4 — TEMPLATE PICKER SYSTEM

When a user clicks "Invoice Generator", "Quotation Generator", or "Receipt Generator", they must first see a Template Picker page before the form.

### Template Picker Page — Layout:
- Heading: "Choose your [Invoice / Quotation / Receipt] Template"
- Subtitle: "All templates support Arabic & English and export to PDF"
- Filter pills: All | Minimal | Corporate | Arabic-First | Compact
- Template cards in a 3-column grid (2 tablet, 1 mobile).

### Each Template Card shows:
- A visual thumbnail preview built using HTML/CSS — a mini scaled-down document with placeholder company header, line items table, total. Scale using CSS transform: scale(0.35) inside an overflow-hidden container sized 200px x 280px.
- Template name below the thumbnail.
- Tags e.g. "Minimal"  "PDF Ready"  "RTL".
- "Use This Template →" button.
- Hover: card border highlights in accent color, "Preview" overlay appears on thumbnail.

### Invoice Templates — 5 options:

1. **Classic** — white background, simple table, logo top-left, client info top-right. Black header text, no colors. Clean and printer-friendly.
2. **Modern Blue** — dark navy header band (#0F172A), white text in header, blue accent on table headers (#2563EB), bold invoice number.
3. **Minimal** — ultra-clean, zero color, hairline borders only, generous whitespace. High-end feel.
4. **Bold Corporate** — deep navy full-width header, company name very large in white, amber accent (#F59E0B) for total row.
5. **Arabic First (عربي)** — RTL layout by default, Arabic heading dominant, Hijri date field, Arabic table headers. Use Cairo font.

### Quotation Templates — 4 options:

1. **Standard Quote** — same structure as Classic Invoice but with validity date, quote number, and acceptance block at bottom.
2. **Proposal Style** — adds a "Project Overview" section at the top before the items table. Wider layout.
3. **Compact Quote** — dense layout, small font, 2-column item table. Good for traders with many items.
4. **Formal Tender** — structured for NGO/government. Fields for Tender Reference, Issuing Authority, Submission Deadline. Formal typography.

### Receipt Templates — 3 options:

1. **Full A4 Receipt** — full-page, company branding, payer details, payment method, reference number, stamp placeholder, signature line.
2. **Thermal POS** — narrow column (320px wide), monospace-style font, minimalist. Looks like a till receipt.
3. **Digital Receipt** — modern card-style, QR code placeholder, colorful header strip, "Payment Confirmed ✓" badge.

### After Template Selection:
- Store selected template in app state (Zustand or React Context).
- Navigate to the form page.
- The form page shows the document preview panel on the right using the selected template — updates live as user fills the form.
- User can click "Change Template" at any point — opens a modal with the template picker without losing form data.
