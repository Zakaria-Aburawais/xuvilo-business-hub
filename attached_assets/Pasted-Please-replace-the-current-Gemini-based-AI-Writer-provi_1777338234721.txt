Validation error:
HTTP 400
{
  "success": false,
  "error": "Invalid request",
  "message": "Please add a main purpose or some key details before generating."

OpenAI API error:
HTTP 502
{
  "success": false,
  "error": "AI provider error",
  "message": "The AI provider could not generate the message. Please try again."
}

Unexpected error:
HTTP 500
{
  "success": false,
  "error": "Server error",
  "message": "Something went wrong while generating the message."
}
