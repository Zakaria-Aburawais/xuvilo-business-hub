Add ZATCA Phase 1 compliant QR code generation to every invoice 
created on Businesses Hub. This is a legal requirement for Saudi 
Arabian businesses and a major feature differentiator.

PART 1 — QR CODE GENERATION ENGINE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ZATCA Phase 1 requires a QR code embedded in every simplified 
tax invoice. The QR code must encode the following 5 fields 
in TLV (Tag-Length-Value) Base64 format:

Tag 1 — Seller Name (string)
Tag 2 — VAT Registration Number (string)
Tag 3 — Invoice Date/Time (ISO 8601 format: YYYY-MM-DDTHH:MM:SSZ)
Tag 4 — Invoice Total (with VAT, 2 decimal places as string)
Tag 5 — VAT Amount (2 decimal places as string)

Implementation steps:

1. Create a utility function: generateZATCAQRCode(invoiceData)
   Input object:
   {
     sellerName: string,
     vatNumber: string,
     invoiceDate: string,       // ISO 8601
     totalWithVAT: number,
     vatAmount: number
   }

2. Encode each field using TLV encoding:
   - Tag: 1 byte (field number)
   - Length: 1 byte (byte length of value)
   - Value: UTF-8 encoded string bytes

3. Concatenate all TLV fields into a single byte array

4. Base64 encode the full byte array

5. Generate a scannable QR code image from the Base64 string
   using the 'qrcode' npm package (qrcode.toDataURL())

6. Return the QR code as a base64 PNG data URL for embedding

PART 2 — INVOICE FORM INTEGRATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Add these fields to the invoice creation form:

1. VAT Registration Number field (optional, shown when 
   country = Saudi Arabia or user enables VAT)
   - Label in English: "VAT Registration Number"
   - Label in Arabic: "الرقم الضريبي"
   - Placeholder: "3XXXXXXXXXXXXXXXXX" (15 digits starting with 3)
   - Add inline validation: must be exactly 15 digits starting with 3

2. Add a "ZATCA Compliant Invoice" toggle switch in the 
   invoice settings panel. When enabled:
   - VAT Registration Number field becomes required
   - QR code is automatically generated and shown in preview
   - Invoice gets a "ZATCA ✓" badge in the top-right corner

3. The QR code should:
   - Appear in the bottom-left of the invoice PDF
   - Be 2cm x 2cm in print size
   - Have the label "Scan to Verify / للتحقق امسح" below it
   - Update in real-time as invoice fields are filled
