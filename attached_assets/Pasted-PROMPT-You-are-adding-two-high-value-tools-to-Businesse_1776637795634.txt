ARABIC TEXT ON CANVAS — TECHNICAL REQUIREMENT
DO NOT use ctx.fillText() for Arabic — it breaks letter joining and produces incorrect glyph shapes.
Use this approach:

Load opentype.js: https://cdn.jsdelivr.net/npm/opentype.js/dist/opentype.min.js
Fetch Cairo font as ArrayBuffer from Google Fonts static CDN
Parse font with opentype.load()
Use font.getPath(text, x, y, fontSize) to get an SVG path object
Draw the path on canvas using path2D.draw(ctx)
For arc text: calculate character positions manually along the arc circumference, render each glyph rotated to follow the arc angle

This is the ONLY correct way to render Arabic on HTML Canvas. Implement it fully.
